# Welcome to the Yonne API

Welcome to the Yonne Developer Hub! Our API allows e-commerce platforms and mobile apps to integrate on-demand delivery logistics across Malawi. 

## 🛠 Prerequisites
Before writing any code, you will need a Yonne Merchant account.
1. Sign up or log in at your Merchant Dashboard.
2. Navigate to **Developer Settings** > **API Keys**.
3. Generate a **Test API Key**.

## 🔐 Authentication
The Yonne API uses API keys to authenticate requests.

**Bearer Token (Recommended)**
`Authorization: Bearer your_api_key_here`

---

## 🚀 Your First API Call
Verify your API key is working by running this cURL command:

```bash
curl --request GET \
  --url [https://api.yonne.app/api/v1/external/validate](https://api.yonne.app/api/v1/external/validate) \
  --header 'Authorization: Bearer yonne_test_your_key_here'
